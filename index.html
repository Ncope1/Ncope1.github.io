Queen Bey Triviamore_vert
Deployed App Github Repo
Queen Bey Triviaclose

The objective of this project was to create a trivia game. I chose to create a six (6) question trivia based on Pop Artist, Beyoncé and her personal/professional life. One question and a set of answer options are displayed when the page renders. The trivia then progresses to the next page when an answer option is selected and a one point score is shown if the question is answered correctly. Once all questions have been answered, “Game Over” is displayed on the final page along with the user’s final score. HTML, CSS and JavaScript were used to create this trivia game.

Deployed App Frontend Repo Backend Repo
The Misadventures of 808's & Dookieclose

This application was created as a group project, and geared toward music lovers who appreciate listening to an artist's or band's entire album to gain a full perspective of the audible impact the artist or band was aiming for. They can then post their own review of the album or create a review for a different album.

The technologies used are: Express, Mongoose, MongoDB, JSON, Heroku, mLab, React, Axios, node-spotify-api npm package, and Bootstrap. Spotify's API was used as well as a Random Name Generator for JavaScript © by Thomas Konings.
